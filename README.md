# 15062026 — TCS Java Practice Problems

A collection of Java solutions for TCS coding practice problems covering algorithms, data structures, and logic building.

---

## 📁 File Overview

| File | Topic | Description |
|------|-------|-------------|
| `Ma.java` | Dynamic Programming | Unique paths in a grid with obstacles (DP approach) |
| `Main1.java` | Logic / Math | Parking fee calculator based on hours parked |
| `Main2.java` | Greedy | Maximum packages that fit within a weight capacity |
| `Main3.java` | Graph / MST | Minimum Spanning Tree using Prim's Algorithm |
| `Main4.java` | Sorting / Matrix | Sort a matrix elements row-wise using Arrays.sort |
| `Main5.java` | Switch-Case | Gym membership fee calculator based on plan months |
| `Main6.java` | Banking Logic | Bank transaction processor with balance validation |
| `Main7.java` | 2D Arrays | Sum of internal and external marks from two matrices |
| `Main8.java` | Sorting / Grouping | Minimize efficiency difference within groups of size T |
| `Main9.java` | Math / Prime | Print all prime numbers in a given range [L, R] |
| `Main10.java` | Math / Speed | Convert distance and time to speed in km/h |
| `Main11.java` | Conditionals | Check if a number is 3-digit and divisible by 9 |
| `Main12.java` | Array / DP | Maximum difference between two elements (buy-sell style) |
| `Main13.java` | Grid / Setup | Grid input setup (base for path problems) |
